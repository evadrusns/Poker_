# Poker
#
# Pour vous mettre dans le contexte fictif, la société Hsh et plus particulièrement le service Direction des systèmes d'Informations souhaite réalisé une maintenance corrective de certains jeux qui se révèle défecteux et présente des réalisations qui n'ont pas de formalités. Nous étions chargés d'améliorer le poker.
# 
# Dans ce README, nous expliquerons dans sa globalité le programme final.
#
# En ce qui concerne les règles, nous avons une main de 5 cartes qui seront générés aléatoirement. Une combiniaison peut alors se révéler, si ce n'est pas le cas ou que l'on souhaite potentiellement l'améliorer alors on peut choisir un nombre de carte que l'on souhaite échanger. Après cela, nous obtenons le résultat de cette main, qui peut être nul ou encore faire partie des combinaisons : paire, double paire, quinte, full, quinte-flush, couleur, brelan et carre.
